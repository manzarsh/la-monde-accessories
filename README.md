# La Monde – Handmade Accessories Website

A luxury-themed responsive web application designed to showcase a handmade accessories brand.

---

## Features
- Elegant dark-themed UI with gold accents  
- Hero section with rotating visuals  
- Collection-based navigation (Designs page)  
- Dedicated pages for Headbands and Brooches  
- Gallery showcase of handmade creations  
- About page with brand storytelling  
- Contact form for custom design requests  
- Fully responsive design  

---

## Tech Stack
- React (Vite)  
- JavaScript (JSX)  
- HTML / CSS  
- Node.js & Express (basic backend structure)  

---

## Project Overview
This project focuses on building a visually engaging and user-friendly interface for a handmade accessories brand.

It highlights:
- Product presentation  
- Brand identity  
- User experience design  

The current version emphasizes frontend development, with backend structure prepared for future expansion.
